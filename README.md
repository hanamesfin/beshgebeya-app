# BeshGebeya Tracker - Python Flask Version

Simple and easy-to-use inventory management system for Ethiopian fresh markets.

## Installation

1. Install Python 3.8+ (if not installed)

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and go to: http://localhost:5000

## First Time Setup

1. Sign up for an account
2. Login
3. Add products
4. Add inventory
5. Record sales
6. Generate alerts
